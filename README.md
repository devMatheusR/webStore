## For local run

Clone the project

```bash
  git clone https://link-para-o-projeto
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  yarn install
```

Inicie o servidor

```bash
  yarn dev
```
